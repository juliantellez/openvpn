# OPENVPN

This repository aims to create base multi-architecture [builds](https://hub.docker.com/r/juliantellez/openvpn/) for [openvpn](https://openvpn.net/).


## Architectures
 - linux/amd64
 - linux/arm64
 - linux/arm/v7
 - linux/arm/v6
